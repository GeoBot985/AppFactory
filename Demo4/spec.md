Project Spec: "NovaCalc" Scientific Calculator1. OverviewA browser-based (or desktop) scientific calculator capable of handling standard arithmetic, trigonometric functions, and logarithmic operations. The focus is on a clean UI and a robust math engine.2. Technical StackFrontend: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).Math Engine: Vanilla JS Math object (or math.js library if complex parsing is needed).3. Core Features & Functional RequirementsFeatureDescriptionBasic OpsAddition, Subtraction, Multiplication, Division.Advanced OpsSquare Root ($\sqrt{x}$), Exponents ($x^y$), Factorials ($n!$).TrigonometrySin, Cos, Tan (toggle between Radians and Degrees).LogarithmsLog (base 10), Ln (natural log).MemoryM+, M-, MR (Memory Recall), and MC (Memory Clear).Input LogicSupport for parentheses () and Order of Operations (PEMDAS).4. UI/UX RequirementsThe Display: A two-line display. Top line shows the full expression (e.g., sin(45) + 10); bottom line shows the live result.Layout: * Right Side: Standard number pad and basic operators.Left Side/Top: Scientific functions (distinctly colored).Responsiveness: Buttons should scale for mobile/desktop.Error Handling: Display "Syntax Error" for invalid inputs (e.g., 5++5 or log(-1)).5. Implementation Roadmap (Tasks for the Model)Phase 1: Build the HTML skeleton and CSS Grid layout.Phase 2: Implement a basic string-based expression parser (Input -> String -> Result).Phase 3: Map scientific buttons to Math.js or native Math functions.Phase 4: Add Keyboard listeners (allow users to type numbers and +, -, etc.).
